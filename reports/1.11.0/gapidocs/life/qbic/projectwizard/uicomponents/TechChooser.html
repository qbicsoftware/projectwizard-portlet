                    • public TechChooser(java.util.List<java.lang.String> options, java.util.Set<java.lang.String> persons)

                      Creates a new condition chooser component

                      Parameters:
                      options - List of different possible conditions
                      other - Name of the "other" condition, which when selected will enable an input field for free text
                      special - Name of a "special" condition like species for the entity input, which when selected will disable the normal species input because there is more than one instance
                      nullSelectionAllowed - true, if the conditions may be empty
